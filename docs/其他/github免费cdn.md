## github使用方法
> 免费好用CDN：jsDelivr+Github 使用方法

### 打tag,release

- **一步搞定** 点击 `release` 发布,或者打 `tag` `push` 即可

使用方法：`https://cdn.jsdelivr.net/gh/你的用户名/你的仓库名@发布的版本号/文件路径`

发布的版本号即 tag 号,不填是最新的。
